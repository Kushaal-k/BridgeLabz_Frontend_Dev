# Web Development Learning Journal - BridgeLabz

---

## Day 1 - 09/10/2025 — Introduction to Web Development & HTML Basics
- Learned: 3-tier architecture (Frontend, Backend, Database); HTML basics (headings, paragraphs, tables, images); VS Code + Live Server; localhost, ports, rendering via IP.
- Task: Create a timetable with `<table>` and view on Live Server.
- Next: HTML block/inline elements, attributes, headings, paragraphs, tables, dropdowns, quotations, lists, blocks, layout.

## Day 2 - 10/10/2025 — Advanced HTML Elements & Forms
- Learned: Block vs inline elements; HTML attributes; headings, paragraphs, tables, dropdowns, lists, layouts; `<marquee>`; HTML forms and input types.
- Task: Build a form with text, email, password, radio, checkbox inputs.
- Next: Begin CSS (styling, look & feel).

## Day 3 - 13/10/2025 — Introduction to CSS
- Learned: CSS syntax and application methods (inline, internal, external); selectors and colors; box model (margin, padding, border, content); display types (inline, block, inline-block); positioning (relative, absolute, fixed).
- Task: Design a simple page using internal/external CSS (backgrounds, margins, padding, positioning).
- Next: Flexbox and Grid for layouts and responsive design.

## Day 4 - 30/10/2025 — Flexbox, Grid & Advanced CSS
- Learned: Flexbox & Grid; advanced box model adjustments; gradients, shadows, transforms; combinators and pseudo-classes.
- Task: Design a calculator layout with CSS Grid, gradients, and box shadows.
- Next: Responsive design, media queries, transitions/animations.

## Day 5 - 31/10/2025 — CSS Pseudo-classes, Animations & Transitions
- Learned: Pseudo-classes (hover, focus, active); tooltips with positioning; CSS animations, transitions, transformations (translate, rotate, scale); box sizing for layouts.
- Task: Create an animation that moves an element diagonally using transitions and transforms.
- Next: Learn Responsive CSS, Filters, and Media Queries.

## Day 6 - 03/11/2025 — Responsive CSS & Media Queries
- Learned: Principles of responsive design; CSS filters (blur, brightness, contrast); media queries for different screen sizes; units (%, vh, vw, em, rem).
- Task: Design a responsive card UI with title, image, text, and hover effects.
- Next: Begin Bootstrap and learn prebuilt responsive utilities.

## Day 7 - 05/11/2025 — Introduction to Bootstrap
- Learned: Bootstrap structure and usage; containers, rows, and columns; 12-column responsive grid system; utility classes for spacing, colors, text; mobile-first design.
- Task: Create a responsive portfolio using Bootstrap Grid and utilities.
- Next: Revise CSS and explore advanced Bootstrap components.

## Day 8 - 06/11/2025 — Bootstrap Components & Responsive Design Revision
- Learned: CSS revision (selectors, positioning, box model); Bootstrap components (carousel, progress bar); combining CSS and Bootstrap; responsive grids & utilities.
- Task: Design a portfolio webpage using Bootstrap with Carousel and Progress Bar.
- Next: Begin JavaScript to make web pages dynamic and interactive.
